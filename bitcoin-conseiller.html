Remarque importante

Le bouton Acheter / Vendre / Attendre n’est pas un vrai conseiller financier. C’est seulement un signal technique automatique basé sur quelques calculs simples.

Le système utilise :

  • RSI ;
  • MACD ;
  • moyennes mobiles 20 / 50 / 200 ;
  • volume ;
  • tendance sur plusieurs périodes ;
  • gestion du risque ;
  • seuils personnalisables ;
  • alerte de prix ;
  • pause et reprise du rafraîchissement ;
  • journal des signaux ;
  • niveau de confiance ;
  • ressources externes ;
  • rafraîchissement dynamique toutes les 20 secondes.
